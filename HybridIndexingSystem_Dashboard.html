Core Innovation

Instead of fully centralizing or fully distributing the index, this system uses a lightweight metadata layer to route queries only to relevant nodes — eliminating unnecessary broadcasts while maintaining distributed scalability.

CENTRALIZED
✗ Single point of failure
✗ Bottleneck at scale
✓ Simple management
DISTRIBUTED

✗ Broadcast overhead
✗ Complex routing
HYBRID (PROPOSED)

✓ Parallel + targeted execution
